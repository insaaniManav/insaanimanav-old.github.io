Make your github profile stand out

Github Profile README

GitHub recently released a feature that allows users to create a profile-level README to display on their GitHub profile. This article will help you to access this new feature.

I’ll also be sharing some fun GitHub profiles I’ve seen so far. If you create yours don’t forget it to share it with me on twitter @insaanimanav .

Why README’s

This feature allows much more than what you can currently do with your bio on github. Since it a readme you can write in markdown as well(did someone say gifs). Another advantage is that it is placed significantly higher than your pinned repositories so it is very clearly visible and will attract attentiom A solid readme in an open source project goes a long way in bringing contributors to it. It also encourages users to use that software. In my opinion this extension of the feature to add a README to your profile will be really nice thing. Also if you are looking to make your project level README’s better here is a good guide.

Where can I get mine

To create your README follow these steps -: Step 1: Create a repo new repo on your github which has the same name as your username. For example my username is insaanimanav so I created a repo called insaanimanav(Note : At the time of writing this the name is case sensitive). Step 2: Create your fancy new README and add stuff to it. Step 3: Commit your readme to your project Step 4: Push your file if you were working locally
